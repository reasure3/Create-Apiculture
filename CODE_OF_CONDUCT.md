# Code of Conduct

## Our Pledge

We want this project to be a welcoming place for mod users, contributors, and maintainers.

## Expected Behavior

- Be respectful and constructive.
- Keep feedback focused on the project and the code.
- Assume good intent when discussing bugs, ideas, and design tradeoffs.

## Unacceptable Behavior

- Harassment, insults, or personal attacks
- Discriminatory language or behavior
- Publishing private information without permission
- Repeatedly derailing issues or pull requests

## Enforcement

Maintainers may edit, hide, or remove comments and may restrict participation when behavior harms the project community.

Instances of unacceptable behavior can be reported through GitHub issues or by contacting the maintainer directly.

A NeoForge mod for Minecraft 1.21.1, built around Create-themed apiculture.

## Requirements

- Minecraft `1.21.1`
- NeoForge `21.1.230` or newer for Minecraft 1.21.1
- Create `6.0.10` or newer in the 6.0.x line
- Kotlin for Forge `5.11.x`

## Building

```powershell
.\gradlew.bat build
```

On Linux or macOS:

```sh
./gradlew build
```

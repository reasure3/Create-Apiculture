# Security Policy

## Supported Versions

Security updates target the latest released version of Create Apiculture.

## Reporting a Vulnerability

Please report security issues privately by opening a GitHub security advisory when available.

If advisories are unavailable, contact the maintainer directly and avoid publishing exploit details until a fix is available.

Include:

- A short description of the issue
- Steps to reproduce
- Affected versions
- Any relevant logs or crash reports
